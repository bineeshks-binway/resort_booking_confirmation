# Wayanad Resort Booking Confirmation Generator

A MERN stack application to generate premium PDF booking confirmations for Wayanad Green Valley Resort.

## Tech Stack
- **Frontend**: Next.js 14, TailwindCSS, TypeScript
- **Backend**: Node.js, Express, Puppeteer
- **Features**: Live Preview, PDF Generation, WhatsApp Integration

## Setup

1. **Server**
   ```bash
   cd server
   npm install
   npm start
   ```

2. **Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```
